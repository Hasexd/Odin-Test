# ODIN

This simple app was made just to test the Odin programming language. It uses Snell's law to allow for zooming in on an image. Changing the index of refraction (IN_OF_REF) to higher values will reuslt in a higher zoom.

# Running
To run the app simply run
```bash
odin run .
```
